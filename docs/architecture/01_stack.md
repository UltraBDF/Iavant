# 01 - Stack Technique & Outils

## 1. Frameworks Coeurs

- **React 18 :** Rendu efficace des UI complexes.
- **Vite 6 :** HMR ultra-rapide et builds optimisés.
- **Tailwind CSS 4 :** Styling moderne avec variables CSS natives.

## 2. 3D & Animation

- **Three.js (R3F) :** Standard pour les graphismes WebGL.
- **Drei :** Helpers pour R3F (chargement GLB, contrôles).
- **Framer Motion :** Orchestration des transitions et animations DOM.

## 3. Primitives UI

- **Radix UI :** Primitives accessibles (Dialog, Tabs) pour WCAG.
- **Lucide-React :** Icônes légères.

## 4. QA & Tests

- **Vitest :** Tests unitaires rapides.
- **Playwright :** Tests E2E pour l'accessibilité.
- **ESLint/Prettier :** Qualité de code.
