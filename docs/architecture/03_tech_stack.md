# 03 - Stack Technique

| Catégorie      | Technologie         | Usage             | Rationale                           |
| -------------- | ------------------- | ----------------- | ----------------------------------- |
| **Framework**  | React 18            | UI & État         | Écosystème riche, rendu performant. |
| **3D**         | Three.js (R3F)      | Scènes immersives | Standard du web pour la 3D.         |
| **Animations** | Framer Motion       | Transitions & DOM | API déclarative puissante.          |
| **Styling**    | Tailwind CSS 4      | Design Système    | Variables CSS natives, rapidité.    |
| **État**       | Zustand             | Réglages N.E.O.   | Simple, rapide, persistant.         |
| **Icons**      | Lucide-React        | Interface         | Léger et cohérent.                  |
| **Build**      | Vite 6              | Bundling          | HMR instantané, builds optimisés.   |
| **Tests**      | Vitest / Playwright | QA                | Compatibilité totale avec Vite.     |
