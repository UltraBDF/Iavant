# Index de la Documentation - L'Internet d'Avant

Bienvenue dans la documentation technique et produit du projet **L'Internet d'Avant (IA)**. Ce projet vise à créer un musée virtuel immersif retraçant l'évolution du Web.

## Plan de lecture

1. **[Brainstorming](brainstorming_session_result.md)** : Genèse et décisions créatives.
2. **[PRD (Product Requirements Document)](prd.md)** : Vision, besoins et fonctionnalités.
3. **[Architecture Document](architecture.md)** : Design système, stack et conventions.
4. **[User Stories](stories/)** : Détails de l'implémentation par épopées (Epics).
5. **[Qualité & Gates](qa/)** : Critères de validation et stratégie de test.
