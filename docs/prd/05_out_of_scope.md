# 05 - Hors Périmètre (V2+)

Les éléments suivants ne font pas partie de la V1 (MVP) mais sont notés pour le futur :

- **Mode Multijoueur :** Possibilité de voir les autres visiteurs dans le musée.
- **Éditeur d'Artefacts :** Permettre aux utilisateurs d'uploader leurs propres souvenirs.
- **Réalité Virtuelle (VR) :** Support des casques VR pour une immersion totale.
- **Tableau de bord admin :** Pour modifier les contenus sans toucher au code.
